Mod Support

EndocZ & Bibliocraft:

Check out the information in the Minecraft(Server) file on Dropbox. The "Finished Product" file has everything needed. Check there and this website for updates regarding our EndocZ mod and texture pack.

Phase III (of EndocZ) includes updates to various blocks, adds colored stone brick, and also implements dragons, via the DragonMounts mod.

Better Then Wolves:

As of September 22nd Endoc server added Better Than Wolves mod. This is a content mod, that adds many features such as basic mechanical power, tanning leather, firing clay pottery, and smelting steel, all of which stay within the basic theme of Minecraft.
