The EndocZ Mod Package

Mods intro:

EndocZ
"EndocZ" is the core mod, written and designed by our very own. Phase V (of EndocZ) has implemented a crucible to melt down all kinds of metals, giant battleaxes and claymores, new bows, and more! Check out the crafting guide for details.

Fleshing out the mod pack are Bibliocraft, Dragons, and NPCs - oh my!

EndocZ_Dragons
This is officially known as the "Dragon Mounts" mod by BarracudaATA.

BiblioCraft
This is the organizational mod created by Nuchaz.

CustomNPCs
The awesome and customizable mod for an RPG-friendly world, by Noppes.
The castle and surrounding areas have been populated with guards, shopkeepers, bandits, and more!

Twilight Forest
Spookey new realm (dimension 7, naturally) filled with new mobs, bosses, mazes, structures, and of course dimly lit forests. Created by Benimatic, forum link here.

The Bits and Pieces:

You can download everything for our mod here. If you have access to the Dropbox folder, you can also find everything there in the "Finished Product 1.7.10" folder. If you have never set up Forge for Endoc before, use the Forge launcher to install Forge client side, run it once, and then a "mods" folder will appear in your ".minecraft" folder (if it doesn't, just make one). EndocZ, EndocZ_Dragons, Bibliocraft, and CustomNPCs will all go in there as is. The Current Resource Pack go in the "resourcepacks" folder in ".minecraft". The resource pack can be found in the Resource Pack tab, and is highly recommended as it has support for all the mods.
